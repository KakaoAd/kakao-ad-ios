KakaoAd SDK 변경 사항

0.8.1

주요 변경 사항

구매 이벤트에 transactionId 파라미터 추가

  • 구매 이벤트(sendPurchaseEvent)에 transactionId 파라미터가 새롭게 추가되었습니다.
    • 주문번호와 같이, 구매에 대한 유니크한 단위 ID를 전달할 수 있습니다.
    • 선택값이며, 리워드 광고 전환 측정이 필요한 경우에 설정합니다.
    • 기존 코드와의 호환성이 유지됩니다. (transactionId에 nil을 전달하면 기존과 동일하게 동작)
    • 구매 이벤트 변경 사항

0.8.0

주요 변경 사항

상품속성 및 이벤트 파라미터 변경

  • KakaoAdDetailProduct 클래스에 brand 프로퍼티가 새롭게 추가되었습니다. 변경 사항 보러 가기
  • 장바구니 추가, 관심상품 추가, 콘텐츠/상품 조회 이벤트의 파라미터가 변경되었습니다.
    • 기존(0.7.8 이하): tag, contentId 전달 방식
    • 변경(0.8.0 이상): tag, currency, products 배열(KakaoAdDetailProduct 활용) 전달 방식 (currency는 ISO4217, 기본값 “KRW”)
    • products는 여러 상품을 배열로 전달하며, brand 등 상세 속성 추가 가능
    • Swift/Objective-C 모두 동일한 방식으로 지원
  • 각 이벤트별 상세 변경 사항 및 예시 코드는 아래 문서를 참고하세요.
    • 장바구니 추가 이벤트 변경 사항
    • 관심상품 추가 이벤트 변경 사항
    • 콘텐츠/상품 조회 이벤트 변경 사항
