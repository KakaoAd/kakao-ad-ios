SDK 설치하기

1) Cocoapods 사용하여 설치

  • 프로젝트의 Podfile에 pod 'KakaoSDK' 를 추가합니다.
  • 터미널에서 pod install 명령을 실행합니다.

2) 수동 설치

  • KakaoSDK를 다운로드 받습니다.
  • KakaoSDK.framework을 앱 프로젝트의 General > Embeded Binaries 항목으로 끌어서 놓습니다.
