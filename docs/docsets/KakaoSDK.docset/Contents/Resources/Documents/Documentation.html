Kakao SDK

이 가이드는 iOS Application 에 이벤트 수집을 위한 가이드를 제공하고 있습니다.

소개

  • iOS 어플리케이션에서 다음과 같은 이벤트를 수집하고 전송하고 있습니다. 수집된 데이터를 이용하여 다양한 목적으로 활용할 수 있습니다.
    • 앱 설치 / 앱 실행 / 가입 완료
    • 장바구니 보기 / 콘텐츠/상품 보기 / 검색
    • 구매 / 앱 내 구매
  • 위의 이벤트 중 앱 설치/ 앱 실행/ 앱 내 구매 이벤트는 별 다른 구현 없이 자동으로 수집 됩니다.

지원환경

  • 최신 버전의 Xcode
  • iOS Deployment Target: iOS 9.0 이상

시작하기

  • 트랙아이디 발급받기
  • SDK 설치하기
  • 프로젝트 설정

연동하기

  • Track ID 설정 및 이벤트 트래커 활성화
  • 가입 완료 이벤트 전송
  • 장바구니 보기 이벤트 전송
  • 콘텐츠 / 상품 조회 이벤트 전송
  • 검색 이벤트 전송
  • 구매 이벤트 전송
